## 红黑树
红黑树，Red-Black Tree 「RBT」是一个自平衡(不是绝对的平衡)的二叉查找树(BST)，树上的每个节点都遵循下面的规则:
* 约束：
	1. 结点可以时红色的或者黑色的
	2. 根结点是黑色的
	3. 叶子结点（空节点）是黑色的
	4. 每个红色的节点的子节点都是黑色的
	5. ::任何一个节点到每一个叶子结点的路径上黑色节点数相同::

* 特点：
	* 趋近于平衡树
	* ::查询的速度非常快::，查询叶子结点的最大次数和最小次数不能超过2倍
